# LeetCode Solutions

This repository contains my solutions to LeetCode problems.

I am solving these exercises regularly to practice problem-solving and not lose coding skills.

All solutions are implemented mainly in C++.

## Structure

Each folder contains a single problem:

```
problem-name/
├── ex.cpp
└── question.txt
```

* `ex.cpp`: solution code
* `question.txt`: problem statement or notes

## Purpose

The main goal of this repository is to stay consistent with algorithm practice and maintain familiarity with common data structures and problem-solving techniques.
